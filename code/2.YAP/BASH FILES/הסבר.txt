התחברות לשרת yapbigmemory בעזרת SSH של GCP.

על מנת להריץ את Yap:
./start_yap
זה מפעיל את יאפ בתור api ברקע ואת הפלט שלו זורק ל /home/guyy18/yapproj/trash

נקרא את trash ונחכה שרשום בסוף Loaded model

עכשיו אפשר להריץ את
python3 seperate_chars.py &

התוכנית לוקחת את כל הdatasets של המיילים והסמסים , מפרידה את הסימנים . , ! ? ומטפלת בסימנים " \ ( ) [ ] { } 
ככה שהיא יכולה להריץ את run_yap עברו כל משפט ולהכניס את כל הפלט לתיקיית output בקובץ מתאים

יש לשים לב בtrash אם יש memory error (זה מפיל את השירות yap ולא ניתן להשתמש בו).
במקרה כזה יש לקחת את קבצי הoutput , לבדוק מה הקובץ האחרון שיצא ומה המשפט האחרון בתוכו,
אם יש קובץ שכל המשפטים בו עברו yap יש לקחת אותו ולרוקן אותו ב datasets (על מנת לא להפעיל עליו שוב את יאפ).
לאחר מכן יש להריץ את התהליך מההתחלה (מ start_yap) ואפילו כדאי לעשות קודם ריסטרט לשרת.
